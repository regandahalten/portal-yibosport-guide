# portal-yibosport-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对某个特定的域名或网站，而是作为通用页面资源的集合，方便统一管理与访问。

## 项目简介

这是一个静态页面归档仓库，主要存放各类自包含的 HTML 文件。每个页面独立存在，不依赖外部数据库或后端服务，适合用于展示说明、工具页面、引导页或临时演示等场景。

仓库结构简单，便于快速部署到静态托管平台。

## 目录说明

```
portal-yibosport-guide/
├── index.html          # 仓库入口页面（可选）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/
│   └── index.html
└── README.md           # 本文件
```

目录结构可根据实际需要灵活调整。每个子目录代表一个独立页面，根目录下的 `index.html` 可作为导航或说明页。

## 页面归档说明

- 所有页面均为纯 HTML + CSS + JavaScript，不包含后端逻辑。
- 页面之间互相独立，互不影响。
- 文件名与路径建议保持简洁，避免特殊字符。
- 若页面包含外部资源（如图片、字体），建议将资源一并归档到对应目录下，确保页面可离线运行。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开任意 HTML 文件即可查看。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）。

## 维护说明

- 新增页面时，建议在根目录下创建新的子文件夹，并将相关文件放入其中。
- 请保持目录结构清晰，避免文件散落在根目录。
- 定期检查页面是否正常显示，尤其是引用了外部资源的页面。
- 本仓库不存储敏感信息，请勿上传包含密码、密钥等内容的文件。
